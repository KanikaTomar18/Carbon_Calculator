Carbon Footprint Calculator

A Flask-based web application that helps users estimate their carbon footprint based on daily activities such as electricity consumption, travel distance, phone usage, and laptop usage.

Features

* Calculate estimated carbon emissions
* User-friendly interface
* Input validation
* Flask backend processing
* Sustainability-focused project
* Real-time result generation

Technologies Used

* Python
* Flask
* HTML5
* CSS3

Project Objective

The goal of this project is to raise awareness about environmental sustainability by helping users understand how everyday activities contribute to carbon emissions.

## Inputs

* Electricity Usage (kWh)
* Travel Distance (km)
* Phone Usage (hours/day)
* Laptop Usage (hours/day)
